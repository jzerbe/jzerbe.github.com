CS2021 Labs - Organization and Architecture was a very interesting course, in that just about all the content of the course (except for expressing oneself in C) was new to me. There were four written assignments and six code assignments (actually five if you don't count reverse-engineering as coding). Of all of the code assignments I would have to say that the performance lab was the most frustrating for me. I was sitting at 27 times the basline speed on a function for the longest time, hitting my head against a wall. I then decided to go crazy breaking abstraction barriers and lots of stuff you should not do in production code work, which landed me 41 times as fast.
