Open the file bin/jvm.config in the root of your SDK install. Update the java.args line to look like java.args=-Xms512m -Xmx1024m -Dsun.io.useCanonCaches=false. Even on one of the latest MacBook Pros with ~2.8x4 GHz and 16GB of RAM, the debugger would barf if I told it to allocate more.

modify Firefox Flash plugin timeout
Enter about:config in the URL bar. The field we are going to modify is called dom.ipc.plugins.timeoutSecs; the default is 45 (seconds) which is not nearly enough of an outing for a decent break point gala event. I like having mine set to 600 (10 minutes).
