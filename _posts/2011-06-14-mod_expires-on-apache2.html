--- layout: post title: mod_expires on apache2 tags: - apache - Server - Web status: publish type: post published: true meta: _edit_last: "1" --- While I was updating the VM that hosts this and several other websites, I decided to give mod_expires (for apache) a try. According to Google's "Page Speed Online" for automated page-load speed testing (analyzes for both Desktop and Mobile devices), I just needed to minify my JS & CSS, and start using the Expires HTTP header on my HTTP server to see increased page load times for my visitors (and less load per user). Fixed with two things:
  1. Following "Make Browsers Cache Static Files With mod_expires On Apache2 (Debian Squeeze)", which essentially comes down to a2enmod expires and fragments for various mime-types, such as: ExpiresByType image/jpg "access plus 60 days"
  2. installing the WP Minify plugin which automates the compression and combining of served JS & CSS
